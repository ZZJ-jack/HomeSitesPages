# 重写项目为网址导航计划

## 项目分析
当前项目是一个基于Vue + Vuetify构建的个人主页，包含以下主要元素：
- 背景（静态/动态壁纸）
- 头像
- 左侧：标签、技能卡片、社交媒体图标
- 右侧：欢迎标题、搜索框、打字机效果、时钟、项目卡片

## 重写需求
1. 保留原有背景、头像、左下角打字效果
2. 删除右边元素以及左侧技能卡片
3. 联系方式放在左下角
4. 新增网址导航功能

## 实现计划

### 1. 修改 App.vue 文件
- 移除右侧的 `homeright` 组件
- 移除左侧的 `polarchart` 组件（技能卡片）
- 移除 `tags` 相关的卡片
- 保留背景视频/图片、头像、打字机效果
- 将社交媒体图标（联系方式）移动到左下角
- 添加网址导航的容器和样式

### 2. 修改 config.js 文件
- 保留原有的背景、头像、打字机配置
- 移除 `polarChart` 配置（技能卡片）
- 移除 `projectcards` 配置（项目卡片）
- 新增 `navigationLinks` 配置，包含分类和链接

### 3. 新增导航链接组件
- 创建一个新的导航链接组件，用于展示分类和链接
- 支持响应式设计，适配不同屏幕尺寸
- 添加链接的悬停效果和点击事件

### 4. 优化样式和布局
- 调整整体布局，使导航链接成为主要内容
- 优化左下角的打字机和联系方式的布局
- 确保响应式设计在移动设备上的良好表现

### 5. 测试和调试
- 测试背景加载是否正常
- 测试头像和打字机效果是否正常
- 测试导航链接的点击是否正常
- 测试响应式布局是否正常

## 技术栈
- Vue 3
- Vuetify 3
- TypeIt（打字机效果）
- 保留原有的依赖，不添加新的依赖
